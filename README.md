# 2026世界杯观赛指南 - 微信小程序

一个功能丰富的2026 FIFA世界杯微信小程序，提供赛程、积分榜、球员数据、日历视图等功能，并支持历年世界杯主题曲循环播放。

## 功能特性

### 1. 首页
- **倒计时组件**: 实时显示距离世界杯开幕的时间
- **快捷入口**: 快速跳转到赛程、日历、球队、球场等页面
- **今日赛事**: 展示当天进行的比赛，支持直播状态显示
- **积分榜**: 
  - 小组积分榜: 12个小组的实时积分排行
  - 总积分榜: 所有参赛球队的综合排名
  - 高级动画: 排名切换时的平滑过渡动画
- **最新资讯**: 世界杯相关新闻动态
- **音乐播放器**: 历年世界杯主题曲循环播放

### 2. 日历页面
- **月历视图**: 清晰展示整个赛事期间的比赛分布
- **日期标记**: 
  - 今天高亮显示
  - 比赛日标记（绿色圆点）
  - 焦点战标记（橙色圆点）
- **每日详情**: 
  - 显示当天所有比赛
  - 自动推荐当日焦点战
  - 点击跳转比赛详情

### 3. 比赛详情页
- **比赛概览**: 比分、时间、球场、天气等信息
- **进球榜**: 双方进球球员及助攻信息
- **助攻榜**: 双方助攻统计
- **阵容数据**:
  - 双方首发球员列表
  - 球员号码、位置、评分
  - 进球、助攻、红黄牌标识
- **比赛统计**:
  - 控球率、射门、射正
  - 角球、犯规、黄牌
  - 传球数、传球成功率
- **事件时间线**: 进球、换人、红黄牌等事件

### 4. 音乐播放器
- **历年主题曲**: 收录1998-2022年世界杯主题曲
- **播放模式**: 列表循环、单曲循环、随机播放
- **唱片动画**: 黑胶唱片旋转效果
- **迷你播放器**: 悬浮在页面底部，不影响浏览
- **全屏播放器**: 完整的播放控制和播放列表

## 技术栈

- **框架**: 微信小程序原生框架
- **样式**: WXSS + CSS3动画
- **音频**: 微信小程序 InnerAudioContext API
- **数据**: 模拟数据（可替换为后端API）

## 界面特色

### 高级感动画效果
1. **倒计时脉冲**: 数字呼吸动画
2. **背景流动**: 足球场背景动态效果
3. **图标浮动**: 快捷入口图标上下浮动
4. **列表入场**: 数据列表逐行滑入动画
5. **排名切换**: 积分榜切换时的平滑过渡
6. **唱片旋转**: 音乐播放器黑胶唱片动画
7. **直播标识**: 呼吸灯效果的直播中标识

### 视觉设计
- 主题色: 足球绿 (#1a5f2a)
- 辅助色: 金色 (#ffd700) 用于高亮
- 渐变背景: 营造层次感
- 圆角卡片: 现代化的UI风格
- 阴影效果: 提升视觉深度

## 使用说明

### 开发环境
- 微信开发者工具
- 小程序基础库 2.19.4+

### 运行步骤
1. 使用微信开发者工具打开项目
2. 填写自己的 AppID
3. 点击编译即可预览

### 图片资源
项目使用以下占位图片路径，请准备相应资源：
- `/images/flags/*.png` - 球队国旗
- `/images/news/*.jpg` - 新闻配图
- `/images/music/*.jpg` - 专辑封面
- `/images/stadiums/*.jpg` - 球场图片
